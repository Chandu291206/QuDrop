Qudrop

A Quantum Key Distribution (QKD) based Secure File Transfer System
BATCH: I17
Year: 2026

Project Overview

  • Qudrop is an end-to-end QKD-based secure file transfer platform.
  • It compares BB84, B92, E91, and Six-State under the same noise settings.
  • The system links key generation, encryption, and transfer in one flow.
BB84 B92 E91 Six-State

Problem Statement

  • Classical key exchange can become vulnerable against quantum-capable attacks.
  • Most protocol demos do not show realistic file-transfer reliability.

Proposed Solution

  • Integrate multi-protocol QKD simulation with practical file transfer.
  • Apply privacy amplification and evaluate QBER/BER per protocol.

Methodology and Use Cases

  1. Alice generates random bits and bases.
  2. Noisy quantum channel is simulated.
  3. Bob performs measurement and key sifting.
  4. QBER is estimated and keys are amplified.
  5. Files are encrypted, transferred, and BER is reported.
Primary deliverables from each run:
1. Protocol-wise QBER comparison table
2. Encrypted/decrypted transfer validation
3. BER and runtime summary metrics
