Qudrop

A Quantum Key Distribution (QKD) based Secure File Transfer System
BATCH: I17
Year: 2026

Project Overview

Qudrop is an end-to-end Python system that simulates Quantum Key Distribution (QKD) for secure file transfer.

It supports multiple protocols to generate shared keys, then uses privacy amplification and XOR encryption for file transmission over TCP sockets.

BB84 B92 E91 Six-State

Problem Statement

  • Classical key exchange methods are vulnerable to interception and future quantum threats.
  • Existing demos often cover only one protocol and do not provide practical transfer + error-analysis insight under channel noise.

Proposed Solution

  • Build a multi-protocol QKD simulation platform integrated with secure file transfer.
  • Generate sifted quantum keys, apply SHA-256 based privacy amplification, and encrypt/decrypt any file format using symmetric XOR logic.

Methodology

  1. Simulate Alice bit/base generation
  2. Simulate noisy quantum channel (bit-flip prob.)
  3. Bob measurement and protocol-wise key sifting
  4. QBER estimation on sifted keys
  5. Privacy amplification (SHA-256)
  6. File encryption/decryption via XOR key stream
  7. Compute BER/Byte Error and timing across trials
